About Pass n Play

Welcome to Pass n Play! This website is a simple 2-player game platform created as a fun way to pass time. While it isn't perfect, it's a great space to enjoy a few classic games with friends. Currently, we offer four games:

  • Tic Tac Toe - A simple but fun game where players take turns marking spaces on a grid to get three in a row.
  • Connect Four - A strategic game where players take turns dropping colored discs into a grid, aiming to get four in a row horizontally, vertically, or diagonally.
  • Flappy Bird - A classic arcade-style game where players control a bird to navigate through pipes.
  • Chess - A traditional strategy game, though currently not fully functioning.

While the website is a work in progress and the games may not be perfect, we hope it provides you with a little bit of entertainment and a way to kill some time. We are constantly improving the site and adding new features, so stay tuned for future updates!

Thank you for visiting Pass n Play!
